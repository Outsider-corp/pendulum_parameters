Вам необходимо подготовить скрипт на языке Python для вычисления параметров маятника по аналогии, как это сделано в Excel (во вложении).
На вход подается информация о реакциях (две вертикальные и одна горизонтальная) в двух опорах сосуда, который частично заполнен жидкостью. После математических преобразований, на выходе получаются три параметра: масса маятника, длина подвеса маятника, высота подвеса.
 
В Excel - желтым цветом выделены ячейки с параметрами, которые необходимо задавать в виде констант.
 
Требования к выполнению задания:
- должен использоваться Python версии не ниже 3.9
- скрипт необходимо написать с применением ООП
- по тексту программы должны быть расставлены комментарии
- к оформлению кода должны быть применены рекомендации PEP 8
- на вход должна поступать таблица в формате .csv, содержащая время и значения реакций (4 столбца)
- входные данные (реакции) должны быть отфильтрованы в диапазоне частот от 0 до 10 Гц
- частота колебаний (ячейка Z36 в Excel) должна вычисляться автоматически
- на выходе должен формироваться файл в формате .csv, содержащий три столбца с параметрами маятника (масса маятника, длина подвеса матника, высота подвеса)
- на выходе должен формироваться файл в формате .csv, содержащий два столбца с промежуточными результатами расчета (значение, размерность, описание)
 
Ожидаемый результат:
- ссылка на пулреквест со скриптом Python в публичном репозитории GitHub
- файл с информацией о виртуальном окружении
- файл .csv, поступающий на вход
 
Обычно, на решение таких задач у студентов уходит 1..2 недели. Однако вы можете корректировать время, учитывая текущую загрузку, связанную с учебой.
